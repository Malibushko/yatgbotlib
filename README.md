# Yet another Telegram Bot Library for C++

Simple lib with a few dependencies (header-only libs included in 'include' folder). 

## Requirments

Compiler with C++17 support 
CMake v 3.5 or higher 

### Build instructions

cd /folder/to/lib
cmake .
make
make install
